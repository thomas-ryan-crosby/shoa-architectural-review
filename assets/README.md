# Assets Folder

This folder contains design assets and reference materials for the Sanctuary HOA Architectural Review Generator.

## Folder Structure

### `logo/`
Place the **Sanctuary logo** file here.

**Requirements:**
- High-resolution logo file
- Formats: PNG (with transparent background preferred) or SVG
- File should be suitable for embedding in PDF documents
- Recommended: PNG at 300 DPI or higher, or SVG vector format

**File naming:** `sanctuary-logo.png` or `sanctuary-logo.svg`

### `reference/`
Place the **styling reference document** here.

This should be an example approval letter that demonstrates:
- Letter formatting and layout
- Typography and font choices
- Spacing and margins
- Logo placement and sizing
- Overall visual styling

**Acceptable formats:**
- PDF (preferred)
- DOCX
- Image files (PNG, JPG) if it's a scanned example

**File naming:** `approval-letter-reference.pdf` or similar descriptive name

---

## Notes

- These files will be used to ensure the generated PDFs match the Sanctuary HOA branding and formatting standards
- The logo will be embedded in the PDF letterhead
- The reference document will guide the layout and styling of generated approval letters
